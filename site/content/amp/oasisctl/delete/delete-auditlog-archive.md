## oasisctl delete auditlog archive

Delete an auditlog archive

```
oasisctl delete auditlog archive [flags]
```

## Options
```
  -i, --auditlog-archive-id string   Identifier of the auditlog archive to delete.
  -h, --help                         help for archive
```
